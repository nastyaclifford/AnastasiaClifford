Website Github
About
The purpose of this project was to build a website accessible on both mobile and desktop devices. It enables users to access information about key nutrients in foods, calculate their daily calorie intake, and discover healthy, balanced recipes. 
Technologies
HTML5, JavaScript, SASS, REST API, GitHub
Project Team
4 front-end developers, 1 team lead
Responsibilities
  • Built a design template in Figma
  • Developed a section of a single page website using responsive web design
  •  Implemented 12 interactive website elements using JavaScript and Edamam API, including recipe finder, user sign- in form, error handling and its display as user-friendly messages
