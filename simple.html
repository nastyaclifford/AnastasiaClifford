About
This project involved creating a game for learning English words, designed to assist people who are studying English by making it easier and faster to memorize new vocabulary. The game allows users to create customized lists of words, and they can update, delete, and edit these lists as needed.
Technologies
React, MobX, JavaScript, SASS, API, GitHub
Responsibilities
  • Designed and developed a single page application with a game for memorizing words
  • Wrote 7 reusable React-based user interface components integrating MobX for state management
  •  Established seamless communication with external APIs, ensuring 99.9% uptime
  • Crafted a user-centric interface with SASS
